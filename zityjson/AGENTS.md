# ZityJSON

This is a zig library for parsing CityJSON files. Including a cli for showing info on a cityjson file and a library with a simple api and bindings to C.

Features:
- Zig version 0.15.2
  - Language spec: https://ziglang.org/documentation/0.15.2/
  - Standard library: https://ziglang.org/documentation/0.15.2/std/

## Development
Using nix. Use the `nix develop` command to enter a shell with all dependencies installed.
